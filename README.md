# AdExt

## Description

AdExt is a browser extension designed to improve your browsing experience if you have blocked certain ad-related sites in your `/etc/hosts` file. Specifically, it addresses issues that arise when the following sites are blocked:

- `*://*.googleadservices.com/*`
- `*://ad.doubleclick.net/*`

By using this extension, you can maintain a smoother browsing experience without the interruptions caused by blocking these sites.

## Installation

This Chrome extension is not published on the Chrome Web Store intentionally. To install it manually from the source code, follow these steps:

1. **Clone or Download the Repository**  
   - Clone the repository using Git:  
   - Alternatively, download the ZIP file from GitHub and extract it.

2. **Open Chrome Extensions Page**  
   - Open Chrome and go to `chrome://extensions/`
   - Enable **Developer mode** (toggle switch in the top right corner).

3. **Load the Extension**  
   - Click on **"Load unpacked"**.
   - Select the folder where you downloaded or cloned this repository.

4. **Use the Extension**  
   - The extension should now be installed and visible in Chromeâ€™s extensions list.
   - Pin it to the toolbar if needed.

To update the extension, pull the latest changes from the repository and click the **Refresh** button on the Chrome extensions page.

## Usage

Once installed, the extension will automatically work in the background to fix any broken browsing experiences caused by the blocked sites.
